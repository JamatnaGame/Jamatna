# هيكل قاعدة البيانات (DB) — قالب جاهز

هذا القالب يجهّز لك مجلد `db/` مقسّم:
- لكل **لعبة** (MCQ) مجلد مستقل
- ولكل **نوع/فرع** ملف JSON مستقل
- مع قوالب خاصة للـ **خلية (hive)** و **الوورد (words)**

## 1) الملفات الأساسية
- `catalog.json` : كتالوج الأنواع والفروع (IDs إنجليزي، أسماء عربي).

## 2) ألعاب MCQ (الكنز/السرعة/احم منطقتك/الثعابين/اختر سؤال للخصم)
المسار:
`db/<game>/mcq/<type>/<sub>.json`

مثال:
`db/treasure/mcq/geo/countries_cities.json`

صيغة الملف:
{
  "type": "geo",
  "sub": "countries_cities",
  "easy":   [ { "q": "...", "a": "...", "o": ["...","...","...","..."], "h": "..." } ],
  "medium": [ ... ],
  "hard":   [ ... ]
}

> ملاحظة: كل ملفات MCQ في هذا القالب تبدأ فارغة (easy/medium/hard = []).

## 3) الخلية (Hive) — الخيار الأول (بنك خاص بالحروف)
المسار:
`db/hive/<type>/<sub>.json`

مثال:
`db/hive/geo/countries_cities.json`

صيغة الملف:
{
  "type": "geo",
  "sub": "countries_cities",
  "easy": {
    "أ": [ { "q":"...", "a":"ألمانيا", "h":"...", "o":["ألمانيا","ألبانيا","أندورا","أيرلندا"] } ],
    "ب": [ ... ]
  },
  "medium": { ... },
  "hard":   { ... }
}

القالب هنا يضيف مفاتيح الأحرف من (أ → م) لكل صعوبة، وكلها فارغة.

## 4) الوورد (Words)
المسار:
`db/words/<type>/<sub>.json`

صيغة الملف:
{
  "type": "islamic",
  "sub": "quran",
  "easy":   [ { "w":"الكوثر", "h":"..." } ],
  "medium": [ ... ],
  "hard":   [ ... ]
}


> **جديد:** تدعم لعبة الوورد الآن الوسائط كسؤال عبر الحقول الاختيارية داخل العنصر:
> - `q` نص السؤال (اختياري)
> - `img` رابط صورة
> - `video` رابط مقطع فيديو
> - `audio` رابط مقطع صوتي
>
> مثال عنصر (من هو؟): `{ "w":"غوكو", "q":"من هو؟", "img":"./assets/goku.jpg" }`

## 5) بدون كلام (Panto) والذاكرة (Memory) — قوالب فقط
- `db/panto/panto.json`
- `db/memory/memory.json`

## 6) أين تضع هذا المجلد؟
ضع محتويات هذا القالب بجانب `index.html` على نفس السيرفر بحيث يكون المسار:
`/db/...`

مثال:
- `index.html`
- `catalog.json`
- `db/treasure/mcq/...`
- `db/hive/...`
- `db/words/...`
